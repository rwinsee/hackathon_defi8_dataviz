# hackathon_defi8_dataviz

Projet dédié au déploiement de l’application R Shiny développée dans le cadre du hackathon « Climat des données ». Ce dépôt contient la configuration nécessaire pour construire l’image (Docker/renv), préparer l’environnement d’exécution et déployer l’application sur le cluster Kubernetes du SSP Cloud via Onyxia (template de service, variables d’environnement, scripts de lancement).
